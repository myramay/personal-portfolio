# ♛ RuPaul's Drag Race Chess ♛

A fully playable chess game in a single HTML file where every piece is a photo of a drag queen — because chess needed more glamour, darling.

## Features

- **Full legal chess rules** — castling, en passant, pawn promotion, check/checkmate/stalemate detection
- **Two sides, two queens:**
  - 💗 White pieces — Trixie & Katya (pink border)
  - 💜 Black pieces — RuPaul (purple border)
- **Emoji badges** to tell piece types apart: 👑 King · ⭐ Queen · 🏰 Rook · 💎 Bishop · 🐎 Knight · 💅 Pawn
- **Click-to-move** with legal move highlighting
- **Campy pink/purple theme** with neon glow effects
- **Self-contained** — images are embedded as base64, no internet required

## How to Play

Just open `rupaul-chess.html` in any browser. No server, no install, no dependencies.

```
open rupaul-chess.html
```

Click a piece to select it (legal moves highlight in green), then click a destination to move. Hit **Sashay Reset!** to start a new game.
